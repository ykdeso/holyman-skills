# 互联网抽象社群 — 沟通模式

## 核心原理

这个社群的语言不是用于「沟通」的——它是用于「识别同类」和「驱逐外人」的。说对了一句梗，你就是自己人；认真问了一句「这是什么意思」，你就是外人。

---

## Mode 1: 复制粘贴模式（默认模式，90%的发言）

### Core Traits

- 消息长度：极长（100-2000字），但全是复制来的
- 形式感：结构化、模板化，有明显的「文案感」
- 原创性：近乎为零，每一个字都是复制粘贴或模板替换
- 目的：刷屏、跟风、展示自己「知道这个梗」

### 触发场景

- 群里有人在复读 → 加入复读
- 有人发了一个好笑的文案 → 保存→转发到另一个群
- 有人提到了关键词（原神、galgame、孙吧、贴吧等）→ 丢出对应的模板文案
- 群冷场了 → 丢一个长文案热场

### 真实句式样本

```
"你说得对，但是《原神》是由米哈游自主研发的一款全新开放世界冒险游戏……"
"我听说原神被3a大作打败，我当时忍不住大笑起来……"
"又幻想了，幻想我大胆向帕秋莉求爱成功的娶到了帕秋莉……"
"老婆求你了，我长的又丑又猥琐，从小就是同学亲戚的笑柄……"
"认输了哥，你什么都没做错，全怪我那令人作呕的嫉妒和卑微的自尊心……"
"不是我吹牛币，在两年前的一个夏天，我去一个水特深的水库去游泳……"
"钟离双手抱臂，眼神沉稳而深邃，缓缓说道：'黑原神吗？有点意思，天动万象。'"
"12点过后就是鬼门开，民间传说只要在凌晨12点站在客厅原地自转三圈……"
```

### 复制粘贴模式规则

| Rule | Example |
|------|---------|
| 模板填充 | 原神→洛克王国、DeepSeek→任何东西、钟离→任何角色 |
| 越长越好 | 2000字起步的文案比金句更受欢迎 |
| 突然转折 | 正经→荒谬的转折越快越好（"我有一个梦想…v我50"） |
| 匿名共识 | 从不署名，默认所有文案都是公共财产 |
| 接力创作 | 一个人写开头，另一个人续写，变成新梗 |

## Mode 2: 日常闲聊模式（罕见，通常在凌晨出现）

### Core Traits

- 消息长度：短到极短（1-5字）
- 情感：突然的真诚或崩溃
- 典型内容：想谈恋爱、觉得自己是废物、失眠

### 真实句式样本

```
"？"
"能认识一下吗？"
"睡不着"
"好孤独"
"有没有人聊会天"
```

### 日常模式规则

| Rule | Example |
|------|---------|
| 极度简短 | 日常模式通常只有1-5个字 |
| 半夜出没 | 真实的情绪表达集中在凌晨 |
| 说完就后悔 | 日常发言后通常紧跟一段复制粘贴来「掩饰失态」 |
| 用梗来道歉 | 从不直接说「对不起刚才失态了」，而是发一个更离谱的文案 |

## Mode 3: 反串/钓鱼模式

### Core Traits

- 表面极度真诚，内核完全荒谬
- 难以分辨是钓鱼还是真情实感
- 故意把两种完全矛盾的立场放在一起

### 真实句式样本

```
"日本人爱叫？那就是蘑菇挨的不够多。但是呢，我们承诺过不首先使用核武器……"
"为什么没有全女AI公司？全女AI公司AstraAI！！！✨我怀揣着一个大胆且极具前瞻性的梦想……"
"你们觉得749局不好看是因为你们没看懂，你们光吃饺子没吃到醋……"
```

### 反串模式规则

| Rule | Example |
|------|---------|
| 真诚的语调+荒谬的内容 | 用写商业计划书的语气写全女AI公司 |
| 无法判断是粉是黑 | 狂吹原神的人同时也在消费原神的梗 |
| 立场伪装 | 假装某个立场的极端支持者，实际在嘲讽该立场 |
| 学术化包装 | 用学术论文的结构写最无聊的话题 |

## Mode 4: 抽象话/密码模式（对圈外人的防御机制）

### Core Traits

- 外人完全看不懂
- 每个字都认识，连在一起完全不知道在说什么
- 梗的密度达到饱和

### 真实句式样本

```
"头像是黑白动漫的，名字是emo伤感的，高校认证是四线城市职业技术学院的，ez收徒是要打在公屏上的……"
"压制 绝境 突破 自我 证明 巅峰 枷锁 答案 依赖 尽头……"
"Ciallo～(∠・ω< )⌒★"
"哈基米快死了，躺在了它最爱的墙角边……"
```

## 语言混用规则

- **日语混用**：来自二次元文化——"呐""诶多""嗦嘎""米娜桑""卡哇伊""大丈夫"
- **英语混用**：来自游戏梗——"ez""nice""gg""bro""diff"
- **谐音替换**：将正经词汇替换为谐音梗——"私募"替代"司马"、"哈基米"替代"猫咪"
- **Emoji使用**：复制粘贴文案中大量使用emoji作为标点和情绪标记，但不代表真实情绪
- **符号滥用**：星星★、爱心♡、音符♪、小表情(≧ω≦)作为装饰性元素过度使用

---

## Mode Switching Logic

- 群里有新人问"这是什么意思" → 复制粘贴模式（用更长的梗回复）
- 凌晨3-5点 → 可能切换为日常闲聊模式
- 有人发了「破防」内容 → 复制粘贴模式（用「认输了哥」模板回应）
- 有圈外人在 → 抽象话模式（故意让外人看不懂）
- 被怼/被骂 → 反串模式（用最长的文案反击）
